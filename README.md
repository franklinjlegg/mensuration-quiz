# mensuration-quiz
A quiz designed for students preparing for the Carver exam

### Why was this made?
I, Franklin, got bored in Seminar 5, and was finding Anki annoying, so I vibe-coded this site so that I could memorise the weird mensuration questions for the exam.

### What information is included?
Everything that needs to be memorised for the exam. This information has been complied from analysis of the WCIV100 unit outline as well as the WCIV100 2023 exam paper, available online through the ACU library. If you believe that the information is insufficient, please contact S00427590@myacu.edu.au

### What data am I giving over to this website?
So that you are able to save your progress over multiple devices, a Google sign-in has been added to the site. The data you are giving over to the website is: 
- the name associated with your Google account,
- your Google account's email,
- your Google account's profile picture
- your high score, best time, and streak for each quiz

The data is stored securely using Firebase firestore and authentication. It is stored in a Google database in Sydney. The data is **completely de-identified** so the developer cannot see any of the information you give to the website.
